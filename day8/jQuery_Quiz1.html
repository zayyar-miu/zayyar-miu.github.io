In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

$ or jQuery is globally avaliable jQuery function. With this, we can create jQuery object. jQuery object is a collection of HTML tags that have been selected by the jQuery function. From this object, we can do different things such as hiding or changing background color or making font size bigger, etc.
var paragraphs = $('p'); Here, $ is a jQuery function and by passing p tag, paragraphs become jQuery object containing p elements inside the page.
paragraphs.hide(); Here, from paragraphs jQuery object, we can use hide method to hide all p elements from the page.
