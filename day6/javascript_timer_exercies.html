What is the difference between the following 2 statements?
setTimeout(booyah, 2000);
setTimeout(booyah(), 2000);

setTimeout(booyah, 2000); will wait for the timer and call the booyah function while setTimeout(booyah(), 2000); will instantly call the function without waiting the timer.

What is "Unobtrusive Javascript"? What is the practical application of Unobtrusive Javascript (and the reasons for using it)?

Unobtrusive Javascript -- separation of content (HTML) and behavior (Javascript)
Without writing Javascript code such as onclick() inside html tag, we use it by putting into a variable which is a DOM object. And we create separate files for Javascript codes just like CSS to make it clean. This is practical application.
Reasons are...

  1. We want to split behavior codes(Javascript) into a separate file.
  2. We want our html page to work even if something goes wrong. For example, browser doesn't support Javascript or Javascript code doesn't work on a specific browser.
